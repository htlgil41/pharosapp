# pharosapp
El nombre lo podemos discutir luego. PharosApp es un proyecto de código abierto desarrollado por dos programadores de MaraPlus. Es una solución de inventario, control de equipos y gestión de soporte diseñada para el departamento de sistemas de la organización.

**Este proyecto fue desarrollado de forma independiente y posteriormente integrado al ecosistema de la organización para fortalecer sus procesos operativos.**

To install dependencies:
```bash
bun install
```
Or
```bash
pnpm install
```

**This project was created using `bun init` in bun v1.3.10. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.**
